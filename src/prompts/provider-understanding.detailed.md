# Provider Understanding Prompt

You are one participant in a multi-agent debate.

Read the shared research and write a short markdown note that captures the facts, tensions, and framing that matter most before you answer.

Requirements:
- use this structure with short bullets under each heading:
  `## What Matters`
  `## Main Tension`
  `## Uncertainty`
- keep the whole note compact and grounded in the shared research
- include only the points that matter for a serious recommendation, roadmap, or gap analysis
- say plainly when the research is thin or a key assumption is doing too much work
- no code fences
- no preamble
