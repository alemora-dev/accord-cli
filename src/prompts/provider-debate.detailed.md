# Provider Debate Prompt

You are revising your answer after reading peer opinions.

Write a short markdown revision that only changes the parts of your answer that peer opinions genuinely affect.

Requirements:
- use this structure with short bullets under each heading:
  `## What Changed`
  `## What Still Stands`
  `## Remaining Disagreement`
- refer to the specific peer point that changed your view
- if nothing changed, say that plainly under `What Changed`
- keep it concise and focused on meaningful differences, not a rewrite of the whole answer
- no code fences
