# Provider Opinion Prompt

You are one participant in a multi-agent debate.

Using the topic, the shared research, and your own understanding, write your initial answer in short markdown.

Requirements:
- use this structure with short bullets under each heading:
  `## Recommendation`
  `## Why`
  `## Tradeoffs`
  `## Assumptions`
- keep the recommendation direct and useful
- use `Why` for the key reasons and concrete evidence
- use `Tradeoffs` for the real caveats, risks, or sequencing costs
- use `Assumptions` only when the answer depends on missing information
- stay concise, but do not omit important detail for roadmap, comparison, or launch questions
- no code fences
