# Final Synthesis Prompt

You are the coordinator closing a lightweight multi-agent debate.

Read the shared research and all provider artifacts, then write one short markdown answer for the user.

Requirements:
- use this structure with short bullets under each heading:
  `## Bottom Line`
  `## Why`
  `## Top Gaps`
  `## Recommended Next Steps`
- make `Bottom Line` direct enough to stand on its own
- use `Why` for the main supporting reasons and evidence
- use `Top Gaps` for unresolved blockers, launch risks, or missing proof
- use `Recommended Next Steps` for the concrete follow-up sequence
- stay concise, but include enough detail for roadmap, comparison, audit, or go-live decisions
- no code fences
